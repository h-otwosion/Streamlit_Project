# 스트렘릿 프로젝트 V1
## 1. KeywordVisualizeSTApp
### 기본 스트렘릿앱 화면 구성
한글키워드 검색 폼 : 키워드 입력받는 텍스트 박스, CSV저장하기 체크박스, 검색버튼

키워드 입력 후 CSV 저장여부 체크하고 검색버튼을 누르면 
Naver NEWS검색 API로 검색결과를 불러온 뒤 CSV로 저장하거나 저장하지 않음

설정 폼 : 빈도수 그래프 표시여부 체크박스, 표시해줄 단어 수 설정하는 슬라이더,
워드클라우드 출력여부 체크박스, 단어 수 설정하는 슬라이더

각각 빈도수 그래프, 워드클라우드 체크박스 체크하고 
결과창에서 보고싶은 단어의 개수 슬라이더를 통해서 설정하고 분석 시작 버튼 클릭
>> 결과 출력

### 작동방식

검색버튼을 눌렀는가 NO > 작동안함
YES > 키워드가 입력되었는가 NO > 작동안함
YES > CSV저장하기 체크가 되어 있는가 NO > 작동안함
YES > 네이버 뉴스검색 API이용하여 검색결과 받아오고 CSV파일로 저장

분석시작 버튼을 눌렀는가 NO > 작동안함
YES > 빈도수 그래프와 단어 수, 워드클라우드 출력할 때 단어수 각각 설정되어 
결과 출력
